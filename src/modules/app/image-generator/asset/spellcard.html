The Image Generator application gives direct access to generating images using SD v1.5, SDXL and SDXL Turbo. The UI allows for generation of images using the same settings across different models making it an ideal tool for comparing the output of different models.The quality and accuracy of the generated images are dependent on the provided negative prompt and settings such as steps, guidance scale, height, and width.

Parameters

Prompt: The prompt to use when generating the image

Negative Prompt: Thing you do not want to see in the generated image.

Steps: Number of steps to take when generating the image

Guidance Scale: Guidance scale to use when generating the image

Height: Height of the image to generate

Width: Width of the image to generate

Seed: Seed to use when generating the latents

# Generations: The number of images you want each model to generate

Models: Models to use when generating the image
